# A.D.A.M V1

A.D.A.M (Autonomous Directive and Arbitration Machine) is an AI-powered assistant system designed to automate responses, manage interactions, and provide adaptive intelligent behavior through modular backend architecture.

## Features

- Automated response handling
- Telegram bot integration
- Rule-based interaction system
- Database-backed storage
- Modular backend structure
- Expandable AI architecture

## Installation

Clone the repository:

```bash
git clone https://github.com/SimplyDraken/A.D.A.M.V1.git
```

Move into the project directory:

```bash
cd A.D.A.M.V1
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running the Project

Run the backend server:

```bash
python adam_project/backend/app.py
```
